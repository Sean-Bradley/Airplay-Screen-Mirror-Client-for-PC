# BoxClient_Demo.exe
## Support System
* Windows XP
* Windows 8
* Windows 10
## Feature
* Send PC Screen to receiver--"ScreenRenderDemo.apk"
* Auto Find receiver
* Support two screen while the pc has two display devices
* Support BJCast
* Support Hardware Encoder
